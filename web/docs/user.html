Mission Control Dashboard

Access the dashboard at https://<your-server>:8080.

  • Governance — View and manage delegative agent boundaries, spending limits, and approval thresholds.
  • Wellness — Monitor financial wellness metrics, spending patterns, and savings rates.
  • Life Stage — Orchestrate multi‑product journeys (home buying, business starting, wealth building).
  • Agent Fleet — Monitor all AI agents, their capability tokens, and activity feeds.
  • ATM Fleet — Manage ATM devices, cash levels, and biometric authentication.
  • Migration — Track legacy core migration progress with parallel‑run validation.
  • Compliance — View regulatory reports, ZK‑proof audit packages, and DORA status.

AI Agents

Verity includes autonomous AI agents that operate within capability‑governed boundaries. Each agent holds a cryptographic identity and cannot exceed its delegated authority. Common agents include Payment Agent, Fraud Detection Agent, and the Verity Companion (customer‑facing financial assistant).
